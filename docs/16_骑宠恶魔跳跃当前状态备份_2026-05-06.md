当前只保留 1 个明确主问题：

1. 开始滑翔后，控制表现是站立姿态；约 1 秒后会闪一下变成跳跃姿态；再过约 1 秒才会完全进入跳跃姿态。目标是让滑翔从开始到结束都保持跳跃姿态。

补充判断：

- 这个问题的性质已经不是“技能未启用”。
- 也不是“封包没发出去”。
- 也不是“根技能 / 子技能映射错了”。
- 当前更像是 `30010186 / afb=362` 的原生动作态、展示态、tail 清理节奏之间没有完全同步。

后续继续修这个问题时，下面这些结论不要再推翻重试：

1. `30010110` 在骑宠态必须继续走 root skill + child skill 的 mounted runtime 路线，不能退回职业原生直放。
